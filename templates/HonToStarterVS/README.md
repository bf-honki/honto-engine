# __HONTO_PROJECT_NAME__

`__HONTO_PROJECT_NAME__` is a Visual Studio starter game for the HonTo SDK static library workflow.

## Open In Visual Studio

1. Install the HonTo SDK or unzip the portable SDK and set `HONTO_SDK_ROOT`.
2. Open `__HONTO_PROJECT_NAME__.sln`.
3. Select `Release | x64` or `Debug | x64`.
4. Build and run `__HONTO_PROJECT_NAME__`.

## What this project shows

- linking against the installed `honto_engine.lib`
- including `honto/HonTo.h` from the SDK
- writing gameplay only in `src/main.cpp`
- using a second tools window from a library-only Visual Studio project
